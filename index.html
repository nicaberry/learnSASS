Как установить SASS:

  1. Открываем консоль;
  2. Устанавливаем Parcel npm install -g parcel-bundler;
  3. Создаём файл package.json для проекта npm init -y;
  4. Устанавливаем parcel index.html для автоматизации приложения;
  5. Устанавливаем Sass npm install -g sass;
  6. Говорим SASS отслеживать все изменения в файле style.scss и вносить их в файл style.css, для этого в консоле вводим sass –watch style.scss:style.css;

Чем крут SASS:

1. Вложенность.

  • Sass позволяет вкладывать стили внутрь других стилей;
  • В вложенности можно исползовать все селекторы CSS;
  • Есть свой селектор &, который заменяет родителя;

Sass позволяет вкладывать стили внутрь других стилей.

Click 
.container-example {
    border: 1px solid grey;
    text-align: center;
    padding: 1rem;
}

.example {
    span {
        background-color: pink;
    }
}

.link {
    color: pink;
    &:hover {
        color: teal;
    }
}

2. Переменные.

  • Sass позволяет создавать переменные;
  • Объявляется переменная с помощью знака $;
  • Далее задаётся любое имя на латинице $myColor;
  • И через двоеточие задаётся значение переменной $myColor: pink;
$border-color: teal;
$accent-color: pink;
$primary-color: grey;
 
.container-example-cub {
    display: flex;
    justify-content: center;
    align-items: center;
}

.example-cub {
    width: 2rem;
    height: 2rem;
    background-color: $primary-color;
    border: 1px solid $border-color;
    margin: 10px;
    &:hover {
        background-color: $accent-color;
    }
}

.cub-1 {
    background-color: $accent-color;
}

3. Импорт.

  • Sass позволяет импортировать данные из другого файла;
  • Для импорта возьмите файл, например button.scss;
  • Далее в отслеживаемом файле в самом вверху документа напишите @import "button.scss(имя/путь импортируемого файла)";
//button.scss
.button-example-import {
    padding: 10px;
    background-color: teal;
    color: rgb(241, 239, 239);
    &:hover {
        background-color: rgb(17, 54, 54);
    }
}

//style.scss
@import "button.scss";

4. Шаблоны.

  • Sass позволяет делать шаблоны;
  • Для объявления шаблона используют знак %(можно заменить на селектор класса, т.е. точку);
  • Далее пишем название шаблона на латинице %firstTemplate(.firstTemplate);
  • Потом открываем фигурные скобки и записываем свойства и значения %firstTemplate(.firstTemplate) { color: pink };
  • Что бы использовать шаблон нужно применить @extend и через пробел написать название шаблона @extend %firstTemplate(.firstTemplate);
%firstTemplate {
    padding: 10px;
    background-color: teal;
    color: rgb(241, 239, 239);
    margin-left: 10px;
}
.button-example-template {
    @extend %firstTemplate;
    &:hover {
        background-color: rgb(17, 54, 54);
    }
}

5. Миксины или примеси.

  • Sass позволяет делать миксины;
  • Для объявления миксина используют @mixin;
  • Далее пишем название миксина на латинице, затем ставим круглые скобки @mixin myFirstMixin();
  • Потом открываем фигурные скобки и записываем свойства и значения @mixin myFirstMixin() { color: pink };
  • В миксины можно передавать переменные @mixin myFirstMixin($colorPink) { color: $colorPink };
  • Что бы использовать миксин нужно применить @include и через пробел написать название миксина и передать значение переменной@include myFirstMixin($colorPink);
@mixin firstMixin($color) {
    background-color: $color;
    padding: 10px;
    color: rgb(241, 239, 239);
    margin-left: 10px;
}
.container-example {
    .mixin-my {
        @include firstMixin(pink);
    }
    .mixin-first {
        @include firstMixin(teal);
    } 
    .mixin-dog {
        @include firstMixin(red);
    }  
    .mixin-mixin {
        @include firstMixin(green);
    }
    .button-example-mixin:hover {
        @include firstMixin(rgb(17, 54, 54));
    }
}

6. Арифметические операции.

  • Sass позволяет использовать арифметические операции: - , + , / , * ;
$fontSBtn: 20px;
.button-example-plus {
    font-size: $fontSBtn;
    width: 100px + $fontSBtn;
}
